# Excel-Project
Developed an Excel model for smart device data using pivot tables and charts to track usage patterns. Cleaned raw data via XLOOKUP, IF formulas, and data validation to ensure accuracy. Built interactive dashboards to visualize key metrics, providing actionable insights and supporting data-driven decision-making for performance trends.
